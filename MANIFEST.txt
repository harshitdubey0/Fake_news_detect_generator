DATA FILES (NOT INCLUDED - Download separately)
======================================================================

To use real datasets, download these files and place in project root:

1. Kaggle Dataset (44,898 articles)
   - https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset
   - Files: Fake.csv, True.csv

2. LIAR Dataset (12,800 statements)
   - https://www.cs.ucsb.edu/~william/data/liar_dataset.zip
   - Files: train.tsv, valid.tsv, test.tsv

3. FakeNewsNet (23,000+ articles)
   - https://github.com/KaiDMML/FakeNewsNet
   - Folders: politifact_fake/, politifact_real/, etc.
